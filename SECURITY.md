# Política de seguridad

## Versiones soportadas

Solo las versiones más recientes reciben actualizaciones de seguridad. Revisá los [releases](https://github.com/idemfede/mcp-server-poncho/releases) para ver qué está soportado.

## Reportar una vulnerabilidad

Si encontrás una vulnerabilidad de seguridad en este proyecto, **no** la abras como issue público.

En su lugar:

1. Enviá los detalles por **email** al mantenedor (revisá el `pom.xml` o el perfil de GitHub del repositorio para datos de contacto), o
2. Usá las [Security Advisories](https://github.com/idemfede/mcp-server-poncho/security/advisories/new) de GitHub si el repo las tiene habilitadas.

Incluí, si es posible:

- Descripción del problema y pasos para reproducirlo
- Impacto esperado (confidencialidad, integridad, disponibilidad)
- Versión afectada y entorno
- Si tenés una idea de mitigación o parche, compartila

Intentaremos responder en un plazo razonable y coordinar la publicación de un fix antes de hacer público el reporte.

## Agradecimientos

Agradecemos a quienes reporten vulnerabilidades de forma responsable y colaboren en la corrección.
